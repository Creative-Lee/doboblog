# 첫 번째 개발 포스트

안녕하세요! 이것은 첫 번째 개발 포스트입니다.

## TypeScript의 장점

TypeScript를 사용하면 다음과 같은 장점이 있습니다:

1. **타입 안전성**: 컴파일 시점에 오류를 발견할 수 있습니다
2. **자동완성**: IDE에서 강력한 자동완성을 제공합니다
3. **리팩토링**: 안전하게 코드를 리팩토링할 수 있습니다

## 코드 예제

다음은 TypeScript로 작성한 간단한 함수입니다:

```typescript
interface User {
  id: number;
  name: string;
  email: string;
}

function getUser(id: number): User {
  return {
    id,
    name: '홍길동',
    email: 'hong@example.com',
  };
}

const user = getUser(1);
console.log(user.name); // 홍길동
```
