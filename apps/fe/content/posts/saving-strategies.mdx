# 20-30대를 위한 저축 전략

## 왜 지금 저축해야 하는가?

20-30대는 **복리의 마법**을 최대한 활용할 수 있는 시기입니다.

### 시간의 가치

```typescript
// 20대부터 저축 시작 (30년)
const earlyStart = 1000 * Math.pow(1.08, 30);
// ≈ 10,063만원

// 40대부터 저축 시작 (10년)
const lateStart = 1000 * Math.pow(1.08, 10);
// ≈ 2,159만원
```

같은 금액을 투자해도 **시작 시기가 빠를수록** 최종 금액이 크게 차이납니다!

## 저축 원칙

### 1. 급여의 30% 저축하기

```
월급 300만원 = 저축 90만원 + 생활비 210만원
```

### 2. 비상금 마련

**최소 6개월치 생활비**를 비상금으로 확보하세요.

```typescript
const monthlyCost = 2_000_000; // 월 생활비 200만원
const emergencyFund = monthlyCost * 6; // 1,200만원
```

### 3. 자동 이체 활용

급여일에 자동으로 저축 계좌로 이체되도록 설정하세요.

## 저축 계획

| 우선순위 | 항목 | 금액 |
|----------|------|------|
| 1 | 비상금 | 6개월치 생활비 |
| 2 | 연금 | 월 급여의 10% |
| 3 | 투자 | 월 급여의 20% |

## 피해야 할 것들

### ❌ 충동구매

불필요한 물건을 사는 습관을 버리세요.

### ❌ 과도한 대출

신용카드 할부와 대출은 최소화하세요.

### ❌ 고위험 투자

전 재산을 한 곳에 투자하지 마세요.

## 실천 팁

1. **가계부 작성**: 지출 내역을 매일 기록하세요
2. **목표 설정**: 구체적인 저축 목표를 세우세요
3. **자동화**: 저축을 자동화하여 의지력에 의존하지 마세요

## 마무리

작은 금액이라도 꾸준히 저축하는 것이 중요합니다.
지금 시작하세요!
