# 투자 기초: 복리의 마법

## 복리란 무엇인가?

복리(Compound Interest)는 이자에 이자가 붙는 것을 의미합니다.

### 단리 vs 복리

**단리 (Simple Interest)**
- 원금에 대해서만 이자가 발생
- 계산이 간단함

**복리 (Compound Interest)**
- 원금 + 이자에 대해 이자가 발생
- 시간이 지날수록 큰 차이 발생

## 복리 계산 공식

```
최종 금액 = 원금 × (1 + 이자율)^기간
```

### 예시

초기 투자금 1,000만원, 연 10% 수익률로 20년 투자 시:

```typescript
const principal = 10_000_000; // 원금 1,000만원
const rate = 0.1; // 연 10%
const years = 20;

const finalAmount = principal * Math.pow(1 + rate, years);
console.log(finalAmount.toLocaleString()); // 약 6,727만원
```

## 시간의 힘

복리 투자에서 가장 중요한 것은 **시간**입니다.

| 기간 | 최종 금액 |
|------|-----------|
| 10년 | 2,594만원 |
| 20년 | 6,727만원 |
| 30년 | 17,449만원 |

## 결론

장기 투자와 복리의 힘을 믿고 꾸준히 투자하는 것이 중요합니다!
